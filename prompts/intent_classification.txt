# Intent Classification
# Regex patterns that classify each user message into a collaboration intent category.
# Used by the "How I Collaborated" section in the report.
#
# Format: category | color | regex pattern
# Split on first 2 pipes only (regex patterns contain | characters).
# The regex is applied to the first 300 characters of each user message.
# If no pattern matches, the message defaults to "Building".
# IMPORTANT: A message can match multiple categories. Iterating is checked
# before Building so overlapping messages (e.g. "also add X") still include
# Iterating first; downstream consumers that use only the first match treat it
# as the primary intent.
